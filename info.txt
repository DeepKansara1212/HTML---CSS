html - HyperText Markup language 

Tag: 
    opening tag: <Hello>
    closing tag: </Hello>

Element: <Hello> Hello World ! </Hello>
<!DOCTYPE html> => Defines the document type as html 

=> There are 2 types of Elements in HTML
    1. Inline Element -> span, img
    2. Block-Level Element -> h1-h6, div, p


=> Lists: Group of related items
    1. Unordered Lists
    2. Ordered Lists
    3. Descriptive Lists



css - Cascading Style Sheets

-> There are 3 ways to add styling to the HTML Elements
    1. Inline css
    2. Internal css
    3. External css 

-> There are 2 selectors in css: 
    1. class -> Access via '.', Multiple Elements can have same class name
    2. id -> Access via '#', ID will be unique
